Schülerliste

Durch diese Option können Sie alle angemeldeten Schüler ansehen. Wenn Sie auf einen bestimmten Schüler klicken, können Sie die Kontaktdaten des Schülers überprüfen und bearbeiten.

Lehrerliste

Durch diese Option können Sie alle Lehrer ansehen. Wenn Sie auf einen bestimmten Lehrer klicken, können Sie die Kontaktdaten des Lehrers überprüfen und bearbeiten. Außerdem können Sie ihre Rechnung hochladen.
